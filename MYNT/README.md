# MYNT双目相机

## SDK安装

```Bash
git clone https://github.com/slightech/MYNT-EYE-S-SDK.git
cd <sdk>
make install
```

### 样例安装

```
cd <sdk>
make samples
```

### ROS安装

```
cd <sdk>
make ros
```

若遇到`cv_bridge`依赖问题，卸载并重装之即可：`sudo apt --reinstall install ros-melodic-cv-bridge`
